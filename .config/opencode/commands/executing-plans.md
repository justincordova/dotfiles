# /executing-plans

Execute a written implementation plan with independent tasks in the current session, with review checkpoints.

## When to Use
- You have a written implementation plan to execute
- Plan has independent tasks that can be worked on in parallel
- Need to execute plan with review checkpoints

## Quick Overview

**Core Principle:** Follow the plan systematically with checkpoints.

**Process:**
1. Read and understand the plan
2. Identify independent tasks
3. Execute tasks in parallel where possible
4. Review progress at checkpoints
5. Verify completion

## Key Principles
- Execute tasks according to the plan
- Use review checkpoints to ensure quality
- Parallelize independent work
- Verify each task completion
