# /subagent-driven-development

Execute implementation plans with independent tasks using multiple subagents working in parallel.

## When to Use
- Have a written implementation plan with independent tasks
- Plan can be broken into parallel work units
- Need to execute plan efficiently with subagents

## Quick Overview

**Core Principle:** Execute independent tasks in parallel using subagents.

**Process:**
1. Read and understand the plan
2. Identify independent tasks
3. Launch subagents for parallel work
4. Collect and integrate results
5. Verify completion

## Key Principles
- Execute tasks according to the plan
- Parallelize independent work using subagents
- Collect and integrate results
- Verify overall completion
